About & How to Use Limud Points

About

Limud Points is a professional teacher management system designed to help teachers manage classrooms, track student points, rewards, and more.

Developed with care for educational excellence. Version 1.1.

How to Use

Getting Started

1. Sign up or log in using your credentials.

2. Create a class or select an existing one from the dropdown.

Managing Students

- Add students individually or in bulk.

- Award or subtract points with reasons.

- Use bulk actions for multiple students.

Rewards and Leaderboard

- Create rewards (points-based or instant).

- View leaderboard for top students.

Admin Features (if applicable)

- View classes and teacher accounts.

- Create new Admins and Teachers.

- Change teacher passwords.

- Add/Remove school-wide embedded tools in the "Extras" section.

For more details, explore the sidebar navigation.

Change Password

To change your password, a password reset link will be sent to your email address.
